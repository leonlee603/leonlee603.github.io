Typing Speed Test

HTML|CSS|JavaScript|AJAX

Typing Speed Test is a page to count the time you need to finish typing the randomly generated text from a JSON file retrieved by AJAX request. The WPM (Word Per Minute) result will be calculated and displayed to user.

F1 Driver 2020 is a search page for getting information of drivers in Formula 1. When user's input are included within the driver's information like name, team or biography, the data shown on the page will be filtered. All the data are stored in JSON and retrieved by AJAX request.
